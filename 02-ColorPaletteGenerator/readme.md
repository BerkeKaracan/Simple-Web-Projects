# 🎨 Color Palette Generator

A modern, one-click web application that generates 5 random HEX color codes and allows instant copying to the clipboard. Built as part of the "100 Projects" series.

## 🚀 Live Demo

Click here to test the application live: [https://simple-web-projects-gdht.vercel.app/]

## ✨ Features

- **Random Color Generation:** Instantly creates new palettes from over 16 million HEX color combinations.
- **One-Click Copy (Clipboard API):** Easily copy any color's HEX code by clicking the copy button below it.
- **Dynamic User Feedback:** Instead of clunky alert windows, an elegant "Copied!" text temporarily appears on the screen, providing a seamless user experience.

## 📱 Design Note (Responsive)

This project was developed with a **Mobile-First** approach. While the UI elements scale fluidly on larger screens without any overflow or layout breakage, the most optimal and intended visual experience is on mobile devices or resized browser windows.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure
- **CSS3:** Flexbox, CSS Grid, gradient backgrounds, and modern UI design
- **Vanilla JavaScript:** DOM manipulation, Clipboard API, custom random HEX algorithm
