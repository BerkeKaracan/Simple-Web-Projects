# 💰 Expense Tracker App

![React](https://img.shields.io/badge/react-%2320232a.svg?style=for-the-badge&logo=react&logoColor=%2361DAFB)
![Vite](https://img.shields.io/badge/vite-%23646CFF.svg?style=for-the-badge&logo=vite&logoColor=white)

A simple and interactive Expense Tracker application built with React and Vite. It allows users to add, delete, and monitor their daily income and expenses. All data is saved in the browser's Local Storage, meaning your transaction history remains intact even after refreshing the page.

## 🚀 Live Demo

**[Click here to view the live project!](https://simple-web-projects-04.vercel.app/)**

## ✨ Features

- **Add Transactions:** Easily add new income (positive) or expenses (negative).
- **Real-time Balance Calculation:** Automatically updates total balance, income, and expense summary.
- **Delete Transactions:** Remove incorrect or old transactions with a single click.
- **Data Persistence:** Uses `localStorage` to save transaction history permanently on the browser.
- **Responsive & Clean UI:** A modern design that is easy on the eyes.

## 🛠️ Built With

- **React.js** (Functional Components, `useState`, `useEffect`)
- **Vite** (Next Generation Frontend Tooling)
- **CSS3** (Flexbox, Custom Styling)
- **Local Storage API**

## 💻 How to Run Locally

If you want to test this project on your local machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/BerkeKaracan/Simple-Web-Projects.git
   ```
2. Navigate to the project directory:
   ```bash
   cd 04-ExpenseTracker
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the Vite development server:
   ```bash
   npm run dev
   ```
