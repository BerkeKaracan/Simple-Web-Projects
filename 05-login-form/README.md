# 🚀 React Local Authentication SPA

This project is a modern Single Page Application (SPA)
for User Registration and Login. It runs entirely in
the browser and uses **Local Storage** as a simulated
database, requiring no backend server.

## 🌟 Key Features

- **⚡ Seamless Routing:** Fast, reload-free navigation between Login and
  Register pages using `react-router-dom`.
- **🛡️ Strict Form Validation:**
  - Enforces all fields to be filled.
  - Validates email format (`@` and `.`).
  - Checks password length and matching confirmation.
- **💾 Local Storage Database:** User credentials are securely saved to and read
  from the browser's local storage in JSON format.
- **🔔 Modern UX:** Uses sleek, UI-integrated error and success
  messages instead of outdated `alert()` dialogs.

## 🛠️ Tech Stack

- **React.js** (UI & State Management)
- **Vite** (Lightning-fast Dev Server & Build Tool)
- **React Router DOM** (Navigation)
- **Vanilla CSS** (Custom Styling)
